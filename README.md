
# .NET CPU Test

Stress test to CPU developed using .NET.

## Installation

This project needs .NET 5 environment to work.

## Running

Clone the project

```bash
  git clone https://github.com/lpsouza/dotnet-cpu-test.git
```

Go to the project directory

```bash
  cd dotnet-cpu-test
```

Start the server

```bash
  dotnet run
```

And watch your CPU on stress!
